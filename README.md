## 安装使用

```
# 克隆项目
git clone https://github.com/heD0ng/front-end-expansion.git

# 客户端部分

# 安装依赖
npm install

# 本地开发 启动项目
npm run dev
```
# 服务端部分

cd errorServer

npm install

nodemon/node index.js
